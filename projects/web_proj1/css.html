CSS Basics

Style your web pages with CSS.

1. Basic Text Styling

h1 {
    color: #333;
    text-align: center;
}

2. Changing Background Color

body {
    background-color: #f4f4f4;
}

3. Adding Borders

div {
    border: 1px solid #ddd;
    padding: 10px;
}

4. Box Model Concept

.box {
    width: 300px;
    padding: 20px;
    border: 2px solid #4CAF50;
    margin: 10px auto;
}

5. Using Flexbox

.container {
    display: flex;
    justify-content: space-between;
}

6. Grid Layout

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
}

7. CSS Hover Effects

button:hover {
    background-color: #4CAF50;
    color: white;
}

8. Pseudo-elements

p::before {
    content: "Note: ";
    font-weight: bold;
}

9. CSS Animations

@keyframes slideIn {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

10. Media Queries for Responsiveness

@media (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}

11. Button Styling

button {
    padding: 10px 20px;
    background-color: #ff6347;
    border-radius: 5px;
}

12. CSS Transitions

a {
    transition: color 0.3s ease;
}

13. Creating Responsive Images

img {
    width: 100%;
    height: auto;
}

14. Adding Shadows

.box {
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

15. Styling Tables

table {
    width: 100%;
    border-collapse: collapse;
}
th, td {
    padding: 10px;
    border: 1px solid #ddd;
}
